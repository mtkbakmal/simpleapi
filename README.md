# SimpleAPI

My little project for learning FastAPI and basic REST architecture.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/mtkbakmal/simpleapi.git
    cd simpleapi
    ```
2. Set up virtual environment:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
3. Install required packages:
    ```bash
    pip install -r requirements.txt
    ```
4. Make sure you have .env file with your database url. There is a .env_example file to your help.
5. Run main.py
    ```bash
    python3 main.py
    ```
6. Visit the "localhost:8000" in your browser.
